# TLÖN Consulting

Sitio web de TLÖN — Estrategia y transformación con IA.

## Estructura

- `index.html` — sitio completo (una sola página, imágenes embebidas en base64).
- `logo.svg` — logo en formato vectorial.
- Archivos `Frame *.png` — assets de diseño de origen.

## Deploy

Publicado en **Vercel** → https://www.tlon-consulting.com

El deploy es automático: cada push a la rama `main` de este repositorio
genera un nuevo despliegue en Vercel.
